# Project M Overview

This project is a tactical turn-based roguelike inspired by games like Advanced Wars, Armored Core, books like The Road
and movies like Elysium.

# Structure

assets/   → raw art, audio, and visual resources  
data/     → gameplay configuration data and stats  
docs/     → design notes and planning documents  
scenes/   → Godot scenes used to build the game  
scripts/  → GDScript code implementing gameplay systems

# Engine

GoDot 4

# Project Goals

- turn-based combat
- customizable mechs
	- body styles, arms, legs and head components
	- multiple weapon classes and damage effectors
- factions that influence playstyle and unique progression trees
- procedurally generated encounters
- multiple map areas that offer unique backdrops that fit encounter types 
- Roguelike progression (whatever that means)
