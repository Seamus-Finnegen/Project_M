# Scripts

This directory contains all GDScript code for the game

Major script groups:

- core: global managers and high-level game state
- run: branching-path and run progression systems
- combat: turn-based combat logic
- mech: mech bodies, parts, weapons, and loadouts
- inventory: storage and equipment management
- scavenging: salvage logic and reward selection
- ui: interface behavior
- utils: shared helper code and constants
