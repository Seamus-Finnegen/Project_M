# Scenes

This directory contains all Godot scene files (`.tscn`)

- main menu scene
	- entry and boot up
- run map 
	- area map -> branching path run scenes
- combat
	- battle scenes, combat objects
- mech bays
	- mech loadout, repair and mechanic/shop scenes
- unit scenes
	- player and enemy mech scenes
- UI scenes
	- reusable interface scenes
- test scenes
	- isolated scenes for testing

Scenes typically reference scripts from the `scripts/` directory and assets from `assets/`

# Note: Scenes are things Godot instantiates and displays
